# 一格电也能kuku生活 (Low Battery, Still Vibing)

专为 Burn-out（精力枯竭）人群设计的生活打卡处。主打“零压力、存在即胜利”。

## 🌟 核心功能

*   **极简生存清单**：只记录最基础的生存目标（如吃饭、喝水、休息），完成即可补充“电量”。
*   **电量反馈系统**：随着基础任务的完成，页面电量从红转黄，最终满电变绿。
*   **小狗垃圾桶**：写下今天消耗你心力的负面情绪，交给小狗“粉碎”吃掉，它会统统带走，只给你留下治愈的拥抱。
*   **备用电池关卡**：当且仅当基础生存指标全部达成时解锁。如果今天还有一点多余的力气，可以去做点额外宠爱自己的小事。
*   **全本地存储与零打扰**：所有数据均保存在你的浏览器本地，没有后端数据库，也没有任何人能看到你的脆弱。
*   **生活记录导出**：支持一键将你所有的生存记录和情绪粉碎历史导出为 CSV 电子表格，留下自己努力生活的痕迹。
